# bin/

Binarios opcionales (llama.cpp / ggml): `.dll` y `.exe` que no forman parte del build de **IA Analyzer**.

- **IA Analyzer** usa `llama-cpp-python` instalado por pip, que trae sus propias librerías.
- Los archivos de esta carpeta pueden eliminarse; el proyecto no los referencia.
- Si los conservás, no se versionan (están en `.gitignore`) para mantener el repo liviano y auditable.
